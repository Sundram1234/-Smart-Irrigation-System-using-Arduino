🌱 Smart Irrigation System using Arduino

🎯 Project Description (हिंदी + English)

यह सिस्टम मिट्टी में नमी की जांच करता है और यदि मिट्टी सूखी होती है तो पानी देने के लिए मोटर/पंप को चालू कर देता है। जैसे ही नमी पर्याप्त हो जाती है, मोटर बंद हो जाती है।

This system detects the soil moisture and automatically turns on the water pump when the soil is dry, and turns it off when the soil has enough moisture.

📦 Required Components:

ComponentQuantity
Arduino Uno1
Soil Moisture Sensor1
Relay Module (1-Channel)1
DC Pump (6V–12V)1
Buzzer (Optional alert)1
Jumper WiresAs needed
9V Battery or Adapter1

⚡ Circuit Connections:

🔌 Soil Moisture Sensor:

  • VCC → Arduino 5V

  • GND → Arduino GND

  • Analog Out (A0) → Arduino A0

🔌 Relay Module:

  • VCC → Arduino 5V

  • GND → Arduino GND

  • IN → Arduino Pin 8

🔌 Pump Connection:

  • Pump का एक terminal 12V battery से

  • दूसरा terminal relay के NO (Normally Open) में

  • Relay का Common (COM) battery के दूसरे terminal से

🧠 Arduino Code:

cpp
int sensorPin = A0;     
int relayPin = 8;       
int threshold = 400;    // Moisture threshold

void setup() {
  pinMode(relayPin, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  int moistureValue = analogRead(sensorPin);
  Serial.print("Moisture Level: ");
  Serial.println(moistureValue);

  if (moistureValue < threshold) {
    digitalWrite(relayPin, HIGH); // Turn ON pump
    Serial.println("Soil is dry – Pump ON");
  } else {
    digitalWrite(relayPin, LOW);  // Turn OFF pump
    Serial.println("Soil is moist – Pump OFF");
  }

  delay(1000);
}
